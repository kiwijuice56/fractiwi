Description

A panel container with basic button functionality such as focusing and pressing.

Functions

  • focus_entered() -> void

    Called automatically when focus is gained. Utilized to update style.

  • focus_exited() -> void

    Called automatically when focus is lost. Utilized to update style.

Variables

  • focused_style: Resource

    Style for when ButtonPanel has focus.

  • disabled_style: Resource

    Style for when ButtonPanel is disabled. Currently unitilized.

  • disabled: bool

    Represents whether ButtonPanel is disabled or not. Currently unitilized.

Signals

  • pressed

    Emit when ui_accept is just pressed and ButtonPanel has focus.
